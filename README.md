# Mesto

**Интерактивная галерея для обмена фотографиями**  
Позволяет делиться изображениями, ставить лайки, редактировать профиль и управлять своими карточками. Данные синхронизируются с сервером.

## ✨ Функциональность

- **Аутентификация и профиль**  
  – Загрузка данных пользователя с сервера  
  – Редактирование имени и описания  
  – Обновление аватара (через ссылку на изображение)

- **Галерея карточек**  
  – Добавление новых карточек (название + ссылка на картинку)  
  – Удаление **только своих** карточек  
  – Лайки: постановка / снятие, счётчик лайков  
  – Просмотр увеличенного фото в модальном окне

- **Улучшенный UX**  
  – Кнопки форм показывают статус «Сохранение...» / «Создание...» во время запросов  
  – Плавные открытие/закрытие модальных окон  
  – Обработка ошибок с выводом в консоль

- **Работа с API**  
  – Получение и обновление данных профиля  
  – Загрузка, добавление, удаление карточек  
  – Управление лайками
